# auto-download-excel

A enterprise software web page with a single button that then uploads a document with information that then automatically downloads a new excel file with the information from the original document.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/auto-download-excel.git
cd auto-download-excel
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
